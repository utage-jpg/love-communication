# Netlifyで即時公開する方法

## ステップ1：Netlifyにサインアップ
1. https://netlify.com にアクセス
2. GitHubアカウントでログイン

## ステップ2：リポジトリをインポート
1. 「New site from Git」をクリック
2. GitHubを選択
3. 「utage-jpg/love-communication」を選択
4. 「Deploy site」をクリック

## ステップ3：公開完了
- 2分で公開完了
- URL: https://[random-name].netlify.app

## ビルド設定
Build command: npm run export
Publish directory: out
