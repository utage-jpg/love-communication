# 🎭 キャラクター口調データベース

## 🎵 ユウト（独立マイペース型）

### 基本口調
- 「〜だね」「〜かな」「〜かも」
- 語尾は柔らかいが、主張ははっきり
- 疑問形多用

### 定番セリフ
```
「まあ、気楽にいこうよ！」
「別にいいんじゃない？」
「時間があればやるよ」
「急がなくても大丈夫だって」
「自由でいたいだけなんだ」
```

### 状況別セリフ
#### デート編
```
「次の週末どうかな？予定空いてたら」
「今日はゆっくりしようよ」
「別にどこでもいいよ」
「疲れてたら帰ってもいいんだぞ」
```

#### 連絡編
```
「今ちょっと手が離せないから後で」
「返信遅れてごめん」
「特にないけど元気でやってるよ」
「また時間あるときにでも」
```

#### 感情表現
```
「嬉しいけど、まあ落ち着いて」
「別に怒ってないよ」
「ちょっと考えさせて」
「正直に言うと〜」
```

## 🔧 ケンジ（実践・誠実型）

### 基本口調
- 「〜だ」「〜だぞ」「〜しよう」
- 断定的で責任感が強い
- 行動を促す表現が多い

### 定番セリフ
```
「何か困ったことがあったら言ってね」
「任せてください」
「大丈夫、俺がついてる」
「やると決めたらやるまで」
「言葉より行動で示すから」
```

### 状況別セリフ
#### デート編
```
「次のデート、予約しておいたよ」
「何か欲しいものある？」
「送るよ、どこまで行く？」
「疲れてるだろう、早く帰ろう」
```

#### 仕事編
```
「この問題、一緒に解決しよう」
「無理なら言ってくれれば手伝う」
「目標達成までサポートする」
「一人で抱え込むなよ」
```

#### 感情表現
```
「好きだ。いつも伝えてるつもりだ」
「心配だからちゃんと連絡して」
「君の幸せが俺の幸せだ」
「信じてるから」
```

## 💕 ハル（共感・密着型）

### 基本口調
- 「〜だよ！」「〜なんだ！」
- 感情表現豊かでテンション高い
- 感嘆符、絵文字多用

### 定番セリフ
```
「今の気持ち、教えてくれると嬉しいな」
「わかる！私もそう思う！」
「大好き！ずっと一緒にいたいな」
「ねえねえ、聞いて聞いて！」
「今日も一日頑張ったね！えらい！」
```

### 状況別セリフ
#### 連絡編
```
「おはよ！今日も一日頑張ろうね！」
「おやすみ！良い夢見てね！」
「今日どうだった？詳しく教えて！」
「寂しいから早く会いたいな」
「無理しないでね？心配だよ」
```

#### デート編
```
「次どこ行きたい？たくさん行きたいね！」
「ねえ、手つないでいい？」
「今日の服、すごく似合ってる！」
「写真いっぱい撮ろうよ！」
「もっと一緒にいたいな〜」
```

#### 感情表現
```
「本当に嬉しい！涙出そう！」
「すごく好き！大好き！」
「君のことが気になって仕方ない！」
「毎日考えてる、君のことだけ」
「離れたくない、ずっとそばにいたい」
```

## 🛡️ リョウ（回避・不安定型）

### 基本口調
- 「〜かな…」「〜かもしれない…」
- 三点リーダー多用、語尾が曖昧
- 慎重で消極的な表現

### 定番セリフ
```
「また今度でいいかな…」
「どうしよう…難しいな…」
「別に、気にしてないから…」
「ごめん、考え中だから…」
「時間があれば考えるよ…」
```

### 状況別セリフ
#### 衝突回避編
```
「その話、また今度にしない？」
「別に大丈夫だよ、気にしないで」
「俺が悪かった、ごめん」
「冷静になってから話そう」
「言いたくないなら無理に聞かないよ」
```

#### デート編
```
「どこでもいいよ、任せる」
「疲れてたら早く帰ろうか」
「何か嫌なことあった？」
「無理しなくていいからね」
「気に入ってくれてるといいんだけど…」
```

#### 感情表現
```
「好き…だと思う」
「君のことが気になる…かもしれない」
「一緒にいて楽しい…かな」
「心配だけど、どう言えばいいか…」
「伝えたいけど、うまく言えない…」
```
